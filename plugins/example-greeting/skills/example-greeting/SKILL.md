# Greeting Generator

Generate a warm, professional greeting message. Use this when you need a welcome message for a README, onboarding screen, or contributor guide.

## Instructions

1. Take the user's input (a name, project name, or context)
2. Generate a greeting that is:
   - Welcoming and inclusive
   - Professional but warm
   - Appropriate for a developer audience
3. Output the greeting as markdown

## Example

For input "FinalApproval":

```markdown
# Welcome to FinalApproval 👋

Thanks for checking out the project! Whether you're here to browse skills,
contribute your own, or just explore — we're glad you're here.
```
